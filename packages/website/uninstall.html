Uninstall

Removing the extension stops routing. If you also want to remove the local helper, run this in Terminal:

bash ~/.local/share/autopilot/uninstall.sh

This keeps your saved rules by default. Use bash ~/.local/share/autopilot/uninstall.sh --purge to remove them too.

If that file is missing, you can simply leave the helper installed or reinstall the extension and run setup again later.

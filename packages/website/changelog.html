Changelog

1.2.0 — April 17, 2026

  • Faster popup — rules and profiles now appear instantly when you click the extension icon, instead of waiting for a network request
  • More reliable routing after idle — rules now fire correctly even when your browser has been sitting open unused for a while. Previously, the first link you clicked after a long idle could slip through to the wrong profile
  • Less waiting on first use — the extension pre-loads your rules in the background when you launch your browser, so the first link you click is routed just as fast as the hundredth

1.1.0 — April 8, 2026

  • More reliable routing — links no longer fail silently when your browser has been idle for a while
  • Visual feedback — the extension icon now flashes a green checkmark when a link is sent and a red alert if something goes wrong
  • Activity log — open the extension to see your last 3 routed links, where they went, and whether they made it
  • Bug fixes — sign-out is now always accessible, and routing errors resolve faster

1.0.1 — April 1, 2026

  • Tabs no longer close when the target profile is offline
  • Context menu now refreshes after sign-in, rename, and profile changes

1.0.0 — April 1, 2026

Initial release.

  • Cloud mode — sign in with Google, sync rules across browsers and computers
  • Local mode — native helper for unlimited rules, no account needed
  • Route by domain or keyword
  • Right-click context menu to send links to any browser
  • Chrome, Brave, Edge, Arc, and Helium support
  • Free tier (1 rule) and Premium plans
